Built like an appliance

Every detail serves one goal: get out of the way and let you write.

  • Real-size pages

    The sheet is drawn at true A4 or 3×4-inch proportions on a textured desk, so the screen previews the printout exactly.

  • Automatic page flow

    Fill a page and the next one appears; delete text and it flows back. On-screen breaks match the exported PDF.

  • Two-colour ribbon

    Black ink for text, brand red for dates in DD MON YYYY — a nod to the old two-colour typewriter ribbon.

  • Export & print

    Export to PDF (cream or white paper) or plain text, generated in pure Python with no third-party libraries.

  • Keyboard-only

    No mouse required. Native shortcuts: ⌘ on macOS, Ctrl on Windows, Linux and the Pi.

  • Truly offline

    No network, no accounts, no telemetry. Files are saved locally as open, human-readable JSON.

Engineering notes

A deliberate exercise in shipping a complete, robust desktop app with zero third-party dependencies.

  • Single-file architecture — one readable Python module.
  • Standard library only — Tkinter for the UI; no installs.
  • Cross-platform — one codebase for macOS, Windows, Linux and Raspberry Pi OS, with a startup guard for the deprecated macOS Tk 8.5.
  • Custom PDF writer — paginated PDFs built by hand, matching the on-screen layout.
  • Security-minded I/O — opened files are sanitised; no eval, no shell, no network.

Stack

  • Python 3
  • Tkinter (Tk 8.6+)
  • JSON document format
  • Hand-rolled PDF generation
  • WAV synthesis (standard library)
